Learning Management System (LMS) Conformance Test

API Implementation Test 2

Status:  Testing in progress...





This test may take several minutes to complete.

Depending upon the functionality of the LMS, this content object may or may not be unloaded automatically upon conclusion of the testing.  When the content object completes, the status above will change to "Test Completed".

This content object tests the LMS for conformance with the SCORM® 2004 4th Edition RTE. Specifically, this content object performs the following:

  • API Implementation Test 2 (visits - 1)
    • Call GetLastError and test for the API to have a starting error code of "0" before Initialize()
    • Test for a successful call to GetErrorString() before Initialize()
    • Test for a call to Terminate() before Initialize()
    • Test for a call to Commit() before Initialize()
    • Test for a call to SetValue() before Initialize()
    • Test for a call to GetValue() before Initialize()
    • Test for a call to Initialize() with an illegal parameter
    • Test for a successful call to Initialize()
    • Test for a call to SetValue() where the data model parameter is an empty characterstring
    • Test for a call to SetValue() where the data model parameter is not a recognized data model
    • Test for a successful call to SetValue()
    • Test for a call to GetValue() where the data model parameter is an empty characterstring
    • Test for a call to GetValue() where the data model parameter is not a recognized data model
    • Test for a successful call to GetValue()
    • Test for a call to Commit() with an illegal parameter
    • Test for a successful call to Commit()
    • Test for a successful call to GetErrorString()
    • Test for a successful call to GetDiagnostic()
    • Test for a successful call to Terminate()
    • Note: Each call to an API function also results in a successful call to GetLastError() to test the return codes.
